## Initialize Envless

Learn ho to initialize Envless in your project.

<Tabs items={["Terminal"]}>
  <Tab>``` envless init ```</Tab>
</Tabs>

Before you run this command, please cd into your project's root directory. This CLI command will create a `.envless` file with project ID and current branch you are working on.

Example:-

```json
{
  "projectId": "xxxxxxxxxxxxxx",
  "branch": "development"
}
```
