## Get started with Envless CLI

Envless CLI works with any programming language, frameworks, hosting platforms and infrastructures, CI/CD pipelines

### Installation

<Tabs items={["npm", "yarn", "brew"]}>
  <Tab>``` npm i -g envless ```</Tab>
  <Tab>``` yarn global add envless ```</Tab>
  <Tab>``` brew install envless ```</Tab>
</Tabs>

### Authentication

Envless CLI provides authentication methods from the command line. Both login and signup happens using the same command.

<Tabs items={["Terminal"]}>
  <Tab>``` envless auth ```</Tab>
</Tabs>

- This will open `/cli/auth` page
- Will ask you for email
- Email you a secure login link
- Signup - If you do not have an existing account, will ask you to create public/private encryption key.
- This private key will be saved securely on your device for other CLI usages.
- Login - If you already have an account, the auth process will complete.
